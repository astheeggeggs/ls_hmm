class lshmm.forward_backward.fb_haploid_variants_samples_tree.CompressedMatrix(ts, normalisation_factor=None)[source]

Class representing a num_samples x num_sites matrix compressed by a tree sequence.

Each site is represented by a set of (node, value) pairs, which act as “mutations”, i.e., any sample that descends from a particular node will inherit that value (unless any other values are on the path).
